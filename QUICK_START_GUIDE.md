- Click **"Zara Example"** button in the top-right

### Step 5: Move to Processes (15 minutes)
- Once 3-5 activities defined, click **"Processes"** tab
- For each activity, define 3 key processes
- Select category from dropdown
- Examples:
  - Real-time sales tracking
  - Weekly team standups
  - Customer feedback collection

### Step 6: Establish KPIs (15 minutes)
- Click **"KPIs"** tab
- For each process, define 3 measurable KPIs
- **CRITICAL:** Must include unit of measurement
  - Examples: 85%, $50K, 7 days, 4.5 score
- Watch for measurability score to climb

### Step 7: Review Dashboard (5 minutes)
- Click **"Dashboard"** tab
- See your complete organizational structure map
- Check metrics:
  - Resource Focus (target: 80%+)
  - Measurability (target: 90%+)
  - Strategic Clarity (target: 85%+)
- Export to PDF or copy summary

### Step 8: Finalize Actions (10 minutes)
- Click **"Actions"** tab
- For each activity, add 3 priority actions:
  - **What:** Specific action
  - **Who:** Responsible person
  - **When:** Deadline date
- Set forum presentation date
- Select your cohort

- Click **"Submit to Guru"** button

## üîß TROUBLESHOOTING

### Problem: Can't Access Next View
**Solution:** Complete all required fields in current view
- Activities: Need 3-5 specific activities
- Processes: Need 3 processes per activity
- KPIs: Need 3 KPIs per process
- Look for lock icon üîí message

### Problem: Validation Says "Too Vague"
**Solution:** Add more context to activity name
- Include HOW you deliver value
- Minimum 15 characters
- Avoid generic terms: Marketing, Sales, Operations

### Problem: KPI Marked as "Not Measurable"
**Solution:** Add unit of measurement
- Select from dropdown: %, $, days, hours, count, score
- Include target value
- Must be objective, not subjective

### Problem: Resource Focus Warning
**Solution:** Concentrate resources in top 5
- Aim for 80%+ total across 5 activities
- Typical split: 25%, 22%, 20%, 18%, 15%
- Remove low-impact activities

### Problem: Lost My Progress
**Solution:** Data auto-saves to browser
- Refresh page - data persists
- Check same browser/device
- Export summary regularly as backup

### Problem: Guru Code Not Working
**Solution:** Check code format
- Must be Base64 encoded
- Copy entire code (no spaces)
- Ensure from same tool version
- Contact Guru if continues
